В якості домашнього завдання необхідно розробити прототип блог системи із використанням фреймоворка CodeIgniter + MySQL + пакет Faker (https://packagist.org/packages/fzaninotto/faker) по прикладу, продемонстрованому викладачем на парі.

Маршрутна карта із переліком пунтків:
====================================
1) спроектувати структуру таблиць бази даних
2) інсталювати фреймворк
3) налаштувати фреймворк
4) встановити бібліотеку faker
5) підключили автозавантаження пакетів з допомогою Composer
6) git commit
7) faker test
8) згенерувати та внести дані в таблицю авторів
9) згенерувати та внести дані в інші таблиці
10) інтегрувати шаблон twitter bootstrap
11) вивести інф. в шаблон
